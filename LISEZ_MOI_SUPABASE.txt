MHUR FRANCE — INSTALLATION SUPABASE COMPLETE

1. Cree un NOUVEAU projet Supabase.
2. Ouvre SQL Editor > New query.
3. Ouvre le fichier INSTALLATION_SUPABASE_COMPLETE.sql.
4. Copie tout le contenu, colle-le dans la requete, puis clique sur Run.
5. Verifie dans Table Editor que les tables communaute et mods existent.
6. Verifie dans Storage que les buckets community-mods et mod-previews existent.
7. Dans le site, ouvre public/js/community-config.js et remplace l'URL et la cle publique par celles du nouveau projet.
8. Utilise uniquement la cle publishable/anon. Ne mets jamais service_role dans le site.
9. Dans Authentication > URL Configuration, ajoute l'adresse publique de ton site comme Site URL et Redirect URL.

Le script est prevu pour un projet Supabase neuf.
